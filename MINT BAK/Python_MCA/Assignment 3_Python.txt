2.Define a function that will calculate the factorial of a number and call the function from the main module with the values ​​taken from the user.

Ans:-
# Function to calculate factorial
def factorial(n):
    fact = 1
    for i in range(1, n + 1):
        fact *= i
    return fact

# Main module
num = int(input("Enter a number to find its factorial: "))

result = factorial(num)
print("Factorial of", num, "is:", result)

3. Define a function that will take the principal amount, time in years and interest rate in percentage as inputs and calculate the simple interest rate. All three arguments have default values ​​- 1000 (principal amount), 5 (time in years) and 10 (interest rate in percentage). Call the function from the main module in all possible ways and calculate the simple interest rate each time.


Ans:-

# Function to calculate simple interest
def simple_interest(principal=1000, time=5, rate=10):
    si = (principal * time * rate) / 100
    return si


# Main module

# 1. Calling with NO arguments (all default values)
print("1. Using all default values:", simple_interest())

# 2. Calling with ONLY principal
print("2. Only principal changed:", simple_interest(2000))

# 3. Calling with principal and time
print("3. Principal and time changed:", simple_interest(2000, 3))

# 4. Calling with all three values
print("4. All three values changed:", simple_interest(2000, 3, 8))

# 5. Calling using keyword arguments (order doesn't matter)
print("5. Using keyword arguments:", simple_interest(rate=12, principal=1500, time=4))

# 6. Calling with only rate changed (keyword)
print("6. Only rate changed:", simple_interest(rate=15))

7. Define a recursive function that will calculate the factorial of a number. Create the program from the main module.

Ans:-

# Recursive function to calculate factorial
def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)


# Main module
num = int(input("Enter a number to find its factorial: "))

result = factorial(num)
print("Factorial of", num, "is:", result)

8. Define a recursive function that will calculate the Fibonacci number for the nth term (the value of n should be input by the user). Create a program from the main module using that function to display the Fibonacci series for n terms.

Ans:-

# Recursive function to calculate Fibonacci of n
def fibonacci(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)


# Main module
n = int(input("Enter how many terms of Fibonacci series you want: "))

print("Fibonacci series:")
for i in range(n):
    print(fibonacci(i), end=" ")

9. Define a recursive function to find the sum of the digits of a number (the number should be input by the user).


Ans:-

# Recursive function to find the sum of digits
def sum_of_digits(n):
    if n == 0:
        return 0
    else:
        return (n % 10) + sum_of_digits(n // 10)


# Main module
num = int(input("Enter a number: "))

result = sum_of_digits(num)
print("Sum of digits of", num, "is:", result)

10. Define a recursive function to calculate the GCD/HCF of two numbers.

Ans:-

# Recursive function to find GCD / HCF
def gcd(a, b):
    if b == 0:
        return a
    else:
        return gcd(b, a % b)


# Main module
x = int(input("Enter first number: "))
y = int(input("Enter second number: "))

result = gcd(x, y)
print("GCD/HCF of", x, "and", y, "is:", result)
